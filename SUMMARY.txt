VERIFICATION STEPS
═══════════════════════════════════════════════════════════════════════════════

After installation, verify everything is working:

1. Check Service Status
   $ ./manage-services.sh status
   [All services should show "active (running)"]

2. Run Health Checks
   $ ./manage-services.sh health
   [All checks should show "✓ is responding"]

3. Verify Systemd Services
   $ sudo systemctl status node-exporter prometheus dbmind
   [All should show "active (running)"]

4. Check Open Ports
   $ sudo netstat -tlnp | grep -E '9090|8080|9100|9187'
   [All ports should be listed]

5. Test Database Connection
   $ gsql -h 192.168.10.153 -p 26000 -U dbmind_monitor -d metadatabase
   [Should connect successfully]

UNINSTALLATION
═══════════════════════════════════════════════════════════════════════════════

To remove everything:

$ sudo ./uninstall-dbmind.sh

Options:
--force    Skip confirmation prompts
--keep-db  Don't remove database user
--help     Show help

The script will:
1. Stop all services
2. Remove systemd service files
3. Delete installed components
4. Clean up temporary files
5. Optionally remove database user
